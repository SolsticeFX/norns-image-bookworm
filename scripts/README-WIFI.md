# norns wifi scripts

These are some very basic scripts intended for the norns software to
manage one wifi connection at a time

## scan
./wifi-scan.sh

## select
../wifi-select.sh "My Wifi SSID" "wifi-password"

## start
./wifi-on.sh

## stop
./wifi-off.sh
